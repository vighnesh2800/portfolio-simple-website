🌐 Portfolio Website

This is a simple personal portfolio website built using HTML and CSS. It includes two main pages: a Home page and a Contact page, designed with a clean and minimal layout.

✨ Features Two-page structure (Home & Contact) Clean and simple user interface Basic navigation between pages Structured layout using HTML and CSS 🛠️ Technologies Used HTML5 CSS3 📄 Pages Home – Introduction and overview Contact – Contact information 🎯 Purpose

This project was created to practice the fundamentals of web development, focusing on building web pages using HTML and styling them with CSS.
